# Required services and supported regions

## Required services

The AWS region must support the following services used in WEKA on AWS.

* AWS EC2 Instances
* AWS Lambda
* AWS Step Functions
* AWS CloudWatch event rule&#x20;
* AWS S3 (required for snap/tiering to object)
